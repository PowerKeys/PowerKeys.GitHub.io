## Win 键增强

> 迅速完成常见 Windows 操作

> 只兼容 Windows

<br>

| 随同 **`Win`** 键按下                    | 功能                                     |
| :--------------------------------------- | :--------------------------------------- |
| **`+`**<br>**`-`**<br>**`\`**            | 增大音量<br>减小音量<br>静音（取消静音） |
| **`[`**<br><big>**`"`**</big><br>**`]`** | 上一首歌<br>暂停（继续）<br>下一首歌     |
| **`Delete`**<br>**`Shift + Delete`**     | 打开回收站<br>清空回收站                 |
| **`End`**                                | 休眠 Windows                             |
